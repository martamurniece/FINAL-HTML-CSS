Plantean

UI design for IOS

This project was created for a client developing a Smart Modular Greenhouse, in a need for an app to conrol the product.

The main goal was to design an IOS app for pre-defined funtionalities and build a coherent and rewarding user experience overall.

Design process included two iteration sessions as well as user testing with stakeholders.
